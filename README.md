# claude code proxy

基于 golang 开发的通用 请求代理程序, 可以方便的查看 api 请求详情, 在使用 `claude code` 是派生此需求, 所以取名为 `ccproxy`

## 安装

## 修改配置文件

配置文件, 可以使用托盘按钮打开, 根据实际请求修改 代理配置

`macos` 

```
~/.ccproxy/config.yaml
```
